Human feedback for Pacman

Instructions


To score actions for our RL Pacman agent:

  1. Look at the current state. Faded sprites show where Pacman and the ghost were in the last two steps.

  2. Hold each arrow button (or hold each arrow key) to set its score 0 (bad) → 1 (good). The longer you hold, the higher the score. Release when happy with the value.

  3. Hold a button again to reset its value and re-score it.

  4. Once happy with all four scores, click 'Submit' or press SPACE. This advances to the next time step.

  5. When an episode finishes a button will appear to start a new episode.

  6. You can skip between time steps using 'Previous' and 'Next'.
